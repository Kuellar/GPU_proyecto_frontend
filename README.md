# Void Seeker

Visit https://icuellar.com/voidseeker/

---
## Development

### Requirements
- [Node](https://nodejs.org/es/): +18.5.0
- npm: +8.12.1

### Install
1. `npm install`
2. `npm run css-build` (apply at every index.scss change)

### Run
1. `npm start`

### Deploy
1. Remove .cache folder
2. Remove dist folder
3. `npm run build-dev`
4. `npm run push-gh-pages`
